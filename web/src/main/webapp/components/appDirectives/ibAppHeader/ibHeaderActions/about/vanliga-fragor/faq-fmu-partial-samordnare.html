Förfrågan

Ja, utredningar kan tilldelas direkt till vårdenheter som drivs i landstingets egen regi.

Genom att klicka på Tilldela direkt, välja den vårdenhet som ska ta emot utredningen och därefter acceptera förfrågan så kommer utredningen tilldelas direkt till vårdenheten.

Kontaktuppgifter och meddelande till Försäkringskassan kommer då att hämtas från vårdenhetens lagrade uppgifter i Intygsbeställning.

Utredning

Utredningstiden baseras på det datum då Försäkringskassan skickat handlingarna. I utredningstiden ingår tid för postgång.

Det datum som registreras som mottagningsdatum av vårdenheten är enbart för den interna uppföljningen och kommuniceras inte till Försäkringskassan.

En utredning avslutas automatiskt när tidsgränsen för Försäkringskassans handläggare att begära komplettering passerats och det inte finns några besök som behöver redovisas. Om det finns besök som behöver redovisas, måste dessa först redovisas och sedan kan vårdadministratören avsluta utredningen manuellt.

Försäkringskassan kan även välja att avbryta en utredning i förtid om exempelvis invånaren uteblir från ett inbokat besöket.

I de fall ett besök måste avbokas på grund av omständigheter som vården ansvarar för och som gör att besöket inte kan genomföras, rapporteras det som en avvikelse orsakad av vården. Utredningen kommer i det fallet att avbrytas och vårdenheten kommer inte bli ersatt.
