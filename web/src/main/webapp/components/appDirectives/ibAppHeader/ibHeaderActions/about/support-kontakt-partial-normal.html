Under sidan "Vanliga frågor" finns hjälp om du har frågor om Webcert. Om du saknar svar på din fråga kan du använda följande kontaktvägar, beroende på typ av fråga:

  • Frågor som gäller inloggning: kontakta leverantören av din e-legitimation eller din SITHS-kortadministratör beroende på hur du loggar in.
  • Om du har problem eller frågor om hur du använder Webcert, eller synpunkter på tjänsten kan du kontakta .
